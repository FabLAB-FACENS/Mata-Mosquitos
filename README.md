# Mata-Mosquitos
		Mata mosquitos customizado

![alt text](https://github.com/FabLAB-FACENS/Mata-Mosquitos/blob/main/ArquivosDeImagem/ProjetoPronto.png)

Inspirado no novo projeto sensação do canal Manual do
Mundo do Iberê Thenório, Nós aqui do Fab resolvemos fazer
nosso próprio mata mosquito customizado.
Estamos fazendo disponibilizando aqui para você que quer fazer
seu próprio mata mosquito no estilo japonês, que além de
limpar sua casa dos pequenos sugadores de sangue, é também
um bonito ornamento de mesa japonês de baixo custo.
